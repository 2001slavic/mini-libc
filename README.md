# Mini libc

A minimally functional implementation of libc. Most of the functions are
implemented using coresponding system calls. 

Implemented functions:
 - `close()`
 - `ftruncate()`
 - `lseek()`
 - `open()`
 - `puts()`
 - `write()`
 - `read()`
 - `truncate()`
 - `malloc()`
 - `calloc()`
 - `free()`
 - `realloc()`
 - `exit()`
 - `sleep()`
 - `nanosleep()`
 - `fstat()`
 - `strcpy()`
 - `strncpy()`
 - `strcat()`
 - `strncat()`
 - `strcmp()`
 - `strncmp()`
 - `strlen()`
 - `strchr()`
 - `strrchr()`
 - `strstr()`
 - `strrstr()`
 - `memcpy()`
 - `memmove()`
 - `memcmp()`
 - `memset()`
